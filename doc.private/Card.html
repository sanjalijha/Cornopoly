Module Card

module Card: sig .. end

Representation of static cards data.

This module represents the data stored in json files pertaning to the monopoly cards, including their description and effect. It handles loading of that data from JSON as well as querying the data.

type card = {
   id : int;
   description : string;
   effect : string * string;
}

The abstract type of values representing a single card.

type t = {
   cards : card array;
}

The abstract type of values representing all cards.
